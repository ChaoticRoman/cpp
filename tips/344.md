* **Did you know that C++26 added `Member visit`?

  * https://wg21.link/P2637

</p></details><details open><summary>Example</summary><p>

```cpp
// C++23
std::visit(overload{
  [](int i){ std::print("i={}\n", i); },
  [](std::string s){ std::print("s={:?}\n", s); }
}, value);

// C++26
value.visit(overload{
  [](int i){ std::print("i={}\n", i); },
  [](std::string s){ std::print("s={:?}\n", s); }
});
```
