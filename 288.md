* **Did you know you can pass an array by reference?**

  * http://eel.is/c++draft/dcl.array

</p></details><details open><summary>Example</summary><p>

```cpp
void foo1(int*);
void foo2(int[42]);
void foo3(int[]);
void foo4(int(&)[42]);

int main() {
    int a[42]{};
    foo1(a); // ok
    foo2(a); // ok
    foo3(a); // ok
    foo4(a); // ok
}
```
