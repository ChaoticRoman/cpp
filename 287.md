* **Did you know that C++23 added `auto(x): decay-copy in the language`?**

  * https://wg21.link/P0849

</p></details><details open><summary>Example</summary><p>

```cpp
auto foo(auto&&);

int main() {
    int i{};
    const int c{};

    foo(42);       // int
    foo(i);        // int&
    foo(int{i});   // int
    foo(auto{i});  // int
    foo(auto(42)); // int
    foo(c);        // const int&
    foo(int{c});   // int
    foo(auto(c));  // int
}
```

  constexpr auto copy_and_sum_ts(auto&... ts); // TODO
